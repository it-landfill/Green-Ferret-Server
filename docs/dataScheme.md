# Data scheme

## Available data

- board id
- latitude
- longitude
- temperature
- humidity
- pressure
- air quality
    - aqi
    - tvoc
    - eco2

### mqtt json label conversion
- lat -> latitude
- lon -> longitude
- tem -> temperature
- hum -> humidity
- pre -> pressure
- aqi -> air_quality_index
- tvo -> tvoc
- eco -> eco2
